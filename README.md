# Dashboard Analítico Pro 🚀

Tablero de control administrativo de alto rendimiento con gráficos interactivos en tiempo real, métricas clave e informes de ventas integrados con Cloudflare D1.

## Características:
- **Gráfico en Tiempo Real**: Visualización dinámica de ingresos mediante Chart.js.
- **Base de Datos D1**: Consultas SQL nativas para inserción, lectura y eliminación de registros.
- **Diseño Premium**: Interfaz oscura con acentos esmeralda, efectos de brillo y diseño responsivo.

## Estructura SQL D1:
- Tabla `sales`: Registra transacciones con campos `id`, `client`, `amount` y `date`.
